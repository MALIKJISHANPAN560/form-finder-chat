# Shan Form Finder

## Project Overview
Shan Form Finder is an AI-powered assistant built using Lovable and Dify.

It helps users find:
- Government forms
- Recruitment forms
- College admission information
- Competitive exam details
- Result-related information

## Features
- AI-powered search
- Knowledge base integration
- Form guidance
- Official links and information
- Student support chatbot

## Technologies Used
- Lovable
- Dify
- Gemini API
- GitHub
